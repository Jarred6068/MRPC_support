This Folder contains code and documents for the Major Updates to MRPC

folder descriptions:
/Simulations/ - contains scripts and .slurm for preforming various simulations of the Updated algorithm

/Simulations/Sim_result_Data/ - contains the simulated data sets and regression results for simulation scenarios for referencing

/Pseudocode Docs/ - contains the pseudocode versions organized by date

/GTEx_tissues_analysis/ - contains analysis of gtex tissues using mrnet_triospecific.R functions for comparison with MRPC

scripts:::

mrnet_general.R - contains the organized codes for the general algorithm
mrnet_triospecific.R - contains the organized codes for the trio specific case
